# Sentiment Analysis on Reviews

## Description
This project uses NLP and Machine Learning to classify reviews as Positive or Negative.

## Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes

## Features
- Text cleaning
- Sentiment prediction
- Machine learning classification

1. Open Google Colab
2. Paste the code
3. Run all cells
